What is acc-set-sible?

set is one of my favorite tabletop games to play with friends and family, but it can be difficult for people with certain kinds of colorblindness to play because the red and green colors may not be discernable.

acc-set-sible starts with a default color scheme that should work for people with many kinds of colorblindness, but also provides the option adjust each color as needed to improve contrast between the colors.

How to Play

The goal of set is to find three cards on the table that make a set. A set is three cards where each individual feature (color, shape, number and shading) is either all the same OR all different!

The makers of set have a good introductory video on how to play to get you started.
