Pick your own colors

How to play

The goal of set is to find three cards on the table that make a set. A set is three cards where each individual feature (color, shape, number and shading) is either all the same OR all different!

The makers of set have a good introductory video on how to play to get you started.

Why acc-set-sible?

Set is one of my favorite tabletop games, but it isn't colorblind friendly. acc-set-sible starts with a default color scheme that should work for people with many kinds of colorblindness, but also provides the option adjust each color as needed. Hopefully, acc-set-sible makes the game available to more people.
